## Additional Notes

### Image URL Format
- **Trimble images** are hosted on Contentful CDN: `images.ctfassets.net/1nvkn1423yot/...`
  - These can be resized by appending query parameters like `?w=800&h=960&fm=webp`
  - The hero product images are PNG format on transparent backgrounds (ideal for web use)
- **NavVis images** are hosted on HubSpot CDN: `3339696.fs1.hubspotusercontent-na1.net/...`
  - Default thumbnails include `?width=300` — remove or increase this for higher resolution
  - Product cutout images (Front/Display views) are 800x800 PNG format
- **Spectra images** are hosted on WordPress: `spectrageospatial.com/wp-content/uploads/...`

### Discontinued Products
- **NavVis M6** — End of support: December 31, 2024. No longer supported by NavVis.
- **Trimble X7** — Appears to be discontinued/superseded by X9. Product page minimal.

### Datasheet Sources
Where official Trimble docushare (trl.trimble.com) links were found, they are noted. Many official datasheets require dealer login or are not publicly linked. Dealer-hosted PDFs (Frontier Precision, Duncan-Parnell, BuildingPoint, GeoNovus, etc.) are reliable alternatives that host the same official Trimble datasheets.
