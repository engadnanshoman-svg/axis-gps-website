# دليل نشر موقع اكسيس على الإنترنت

## الطريقة الأسهل: Vercel (مجاني وموصى به) ✅

### الخطوة 1: إنشاء حساب GitHub
1. اذهب إلى **https://github.com** واضغط **Sign up**
2. سجّل بالبريد الإلكتروني وكلمة مرور
3. اختر الخطة المجانية (Free)

### الخطوة 2: رفع المشروع على GitHub
1. بعد تسجيل الدخول، اضغط **"+"** ثم **"New repository"**
2. اكتب اسم المستودع: `axis-gps-website`
3. اختر **Private** (خاص) أو **Public** (عام)
4. اضغط **"Create repository"**
5. اضغط **"uploading an existing file"**
6. اسحب جميع ملفات المشروع (من ملف ZIP المرفق) وأفلتها
7. اضغط **"Commit changes"**

### الخطوة 3: النشر على Vercel
1. اذهب إلى **https://vercel.com** واضغط **"Sign Up"**
2. اختر **"Continue with GitHub"** واربط حسابك
3. اضغط **"Add New"** → **"Project"**
4. اختر مستودع `axis-gps-website`
5. اضغط **"Deploy"** - خلال دقيقة واحدة سيكون الموقع متاحاً! 🎉

### الخطوة 4: ربط الدومين (اختياري)
1. في لوحة تحكم Vercel → **Settings** → **Domains**
2. أضف الدومين: `www.axis-gps.com`
3. اذهب إلى مزود الدومين وعدّل سجلات DNS:
   - أضف سجل **CNAME** يشير إلى `cname.vercel-dns.com`
4. انتظر حتى يتم التفعيل (قد يستغرق حتى 24 ساعة)

## الطريقة الثانية: Netlify (مجاني أيضاً)

1. اذهب إلى **https://netlify.com** وأنشئ حساباً
2. اسحب مجلد المشروع وأفلته في منطقة النشر
3. الموقع سيكون متاحاً فوراً

## الطريقة الثالثة: استضافة مدفوعة (للمحترفين)

يمكنك استئجار خادم VPS من شركات مثل:
- **DigitalOcean** (5$/شهر)
- **Hetzner** (3.5 يورو/شهر)
- **AWS EC2** (حسب الاستخدام)

ثم رفع الملفات وتشغيل:
```bash
npm install
npm run build
npm run start
```
