# 🚀 دليل نشر موقع اكسيس على الإنترنت

## موقع اكسيس للحلول الهندسية المتقدمة - AXIS GPS

---

## الطريقة الأسهل: النشر عبر Vercel (موصى بها) ⭐

Vercel هو أفضل منصة لنشر مواقع Next.js مجاناً وبدون حدود عملية.

### الخطوة 1: إنشاء حساب GitHub
1. اذهب إلى **https://github.com/signup**
2. أنشئ حساباً مجانياً (اسم المستخدم + البريد الإلكتروني + كلمة المرور)
3. أكد البريد الإلكتروني

### الخطوة 2: رفع المشروع إلى GitHub
1. بعد تسجيل الدخول في GitHub، اضغط على **"New repository"**
2. سمّ المستودع: **axis-gps-website**
3. اختر **Private** أو **Public**
4. اضغط **"Create repository"**
5. ارفع ملف ZIP المضغوط باستخدام موقع:
   - اذهب إلى **https://github.com/your-username/axis-gps-website**
   - اضغط **"uploading an existing file"**
   - اسحب ملف **axis-gps-website.zip** وافتحه أولاً، ثم ارفع جميع الملفات
   - أو استخدم الأوامر التالية في Terminal:

```bash
cd axis-gps-website
git init
git add .
git commit -m "نشر موقع اكسيس"
git remote add origin https://github.com/YOUR-USERNAME/axis-gps-website.git
git push -u origin main
```

### الخطوة 3: النشر على Vercel
1. اذهب إلى **https://vercel.com/signup**
2. سجّل باستخدام حساب GitHub (مجاني!)
3. بعد تسجيل الدخول، اضغط **"Add New Project"**
4. اختر مستودع **axis-gps-website**
5. اترك الإعدادات الافتراضية (Framework: Next.js)
6. اضغط **"Deploy"**
7. ✅ خلال دقيقة واحدة، سيكون موقعك متاحاً على الإنترنت!

### الخطوة 4: ربط دومين مخصص (اختياري)
1. من لوحة تحكم Vercel، اذهب إلى **Settings → Domains**
2. أضف دومينك: **axis-gps.com**
3. اتصل بمزود الدومين لتغيير سجلات DNS:
   - أضف سجل CNAME: `www` → `cname.vercel-dns.com`
   - أو سجل A: `@` → `76.76.21.21`

## الطريقة البديلة: Netlify

### الخطوات:
1. اذهب إلى **https://netlify.com/signup**
2. سجّل باستخدام GitHub
3. اضغط **"Add new site" → "Import an existing project"**
4. اختر مستودع **axis-gps-website**
5. إعدادات البناء:
   - Build command: `npm run build`
   - Publish directory: `out`
6. اضغط **"Deploy site"**

---

## الطريقة الثالثة: Cloudflare Pages

1. اذهب إلى **https://dash.cloudflare.com/sign-up**
2. أنشئ حساباً مجانياً
3. اذهب إلى **Workers & Pages → Create application → Pages**
4. اختر **"Connect to Git"** واربط GitHub
5. اختر المستودع وإعدادات:
   - Build command: `npm run build`
   - Build output directory: `out`
