    5. التقنيات المستخدمة
================================================================================

🔧 الإطار: Next.js 16 (App Router)
🎨 التصميم: Tailwind CSS 4 مع ألوان OKLCH مخصصة
🎭 الحركات: Framer Motion (scroll reveal, counters, floating elements)
🗺️ الخرائط: Leaflet + react-leaflet + OpenStreetMap
📱 التجاوب: تصميم متجاوب بالكامل (جوال/تابلت/كمبيوتر)
🔤 اللغة: عربية RTL بالكامل
🖼️ الأيقونات: Lucide React
🌙 الثيم: داكن هندسي مع لمسات teal/cyan

================================================================================
    7. خطوات نشر الموقع على الإنترنت
================================================================================

🟢 الطريقة الأسهل: Vercel (مجاني وموصى به لـ Next.js)

الخطوة 1: إنشاء حساب على GitHub
   - اذهب إلى https://github.com وإنشاء حساب مجاني

الخطوة 2: رفع المشروع على GitHub
   - إنشاء مستودع جديد (Repository)
   - رفع ملفات المشروع

الخطوة 3: النشر على Vercel
   - اذهب إلى https://vercel.com وإنشاء حساب
   - اربط حساب GitHub
   - اختر المستودع واضغط "Deploy"
   - خلال دقيقة واحدة سيكون الموقع متاحاً!

الخطوة 4: ربط دومين مخصص (اختياري)
   - في لوحة تحكم Vercel > Settings > Domains
   - أضف الدومين: www.axis-gps.com
   - عدّل سجلات DNS عند مزود الدومين
